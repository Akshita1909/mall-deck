# Dubai Mall Interactive Sales Deck

## Overview
Interactive web-based sales deck prototype showcasing Dubai Mall as a global retail, entertainment, and partnership destination.

## Features
- Cinematic video-first landing
- Interactive sections (Stats, Attractions, Retail)
- Business-focused CTA for leasing & partnerships
- Smooth animations (Framer Motion)

## Tech Stack
- React + Vite
- Framer Motion
- CSS

## Setup
```bash
npm install
npm run dev
```

## Deployment
Deploy via Vercel or Netlify.

## Notes
Designed as a scalable architecture for future modules like Events, Sponsorships, and Leasing flows.
